# Project 2 - E-Commerce Homepage Wireframe

## Overview

This project was completed as part of the DecodeLabs UI/UX Design Internship (Batch 2026).

The objective of this exercise was to create a low-fidelity wireframe for an e-commerce homepage. The focus was not on colors or visual design, but on organizing content in a way that makes it easy for users to browse products, discover categories, and navigate the website efficiently.

The challenge was to create a homepage layout that feels clear, intuitive, and user-friendly while avoiding unnecessary clutter.

## Project Goal

The purpose of this project was to design a homepage structure that helps users quickly understand what the store offers and easily access important sections such as categories, featured products, promotions, and best-selling items.

## What I Did

- Designed a desktop homepage wireframe
- Created a clear navigation structure
- Organized content using visual hierarchy principles
- Added sections for product categories, featured products, and best sellers
- Included promotional banners and call-to-action buttons
- Designed a layout focused on usability and discoverability

## Key Insight

One of the main lessons from this project was understanding how layout and content placement influence user behavior. Even without colors or high-fidelity visuals, a well-structured wireframe can guide users through a page and help them find information more efficiently.

## Why This Matters

A homepage is often the first interaction users have with an online store. A confusing layout can make users leave quickly, while a clear and organized structure helps users browse confidently and encourages engagement.

This exercise reinforced the importance of information architecture and user-centered design before moving into visual design.
